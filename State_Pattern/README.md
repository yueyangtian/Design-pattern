## 状态模式

#### 模式动机

* 允许对象在内部状态发生改变时改变它的行为，对象看起来好像修改了它的类。主要对对象的行为依赖于它的状态（属性），并且可以根据它的状态改变而改变它的相关行为进行抽象。

#### 模式结构

* 以一个投票类为例，我们对是否已经投票与投票当一个用户已经投过票状态与投票行为进行抽象
